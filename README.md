## 🚀 Deploy on GitHub Pages

```bash
git init
git add .
git commit -m "The Stationery Hub — initial commit"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/stationery-hub.git
git push -u origin main
```

Then go to: **GitHub repo → Settings → Pages → Source: main branch → Save**

Your site will be live at:
`https://YOUR_USERNAME.github.io/stationery-hub/`

## ✏️ Things to Customise

Open `index.html` and find + replace these:

| Placeholder | Replace with |
|---|---|
| `+91 XXXXX XXXXX` | Your real phone number |
| `91XXXXXXXXXX` | Your WhatsApp number (numbers only) |
| `Main Market Road, Your City — 700 001` | Your real address |
| `hello@thestationeryhub.in` | Your email (if added) |
| `href="#"` in footer socials | Your Facebook / Instagram links |
