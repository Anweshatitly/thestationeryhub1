## 🚀 Deploy on GitHub Pages

```bash
git init
git add .
git commit -m "Initial commit — The Stationery Hub"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/stationery-hub.git
git push -u origin main
```

Then: **GitHub → Settings → Pages → Source: main → Save**

Live at: `https://YOUR_USERNAME.github.io/stationery-hub/`

## ✏️ Customise

| Phone number | `index.html` → search `XXXXX XXXXX` |
| Address | `index.html` → contact section |
| WhatsApp link | `index.html` → `btn-whatsapp` href |
| Store hours | `index.html` → contact section |
| Colors | `css/style.css` → `:root` variables |
